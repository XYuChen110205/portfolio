## 六、常见问题

### Python环境
- 建议使用 **Python 3.11+**（部分项目需要3.10+/3.8+）
- 使用虚拟环境隔离不同项目的依赖：`python -m venv venv`
- CUDA版本的PyTorch：`pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121`

### Node.js环境
- 推荐 **Node.js 18+**
- 如遇pnpm未安装：`npm install -g pnpm`

### Docker环境
- Film Genome System 和 BGF 需要Docker
- Windows使用 Docker Desktop

### 端口冲突
- 如遇端口被占用，检查并关闭已运行的服务
- 常用端口：3000, 4000, 5000, 5020, 5173, 5174, 8000, 8001
